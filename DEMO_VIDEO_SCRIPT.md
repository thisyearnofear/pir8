## üé• Recording Tips

### Setup
1. **Screen Resolution**: 1920x1080 or 1280x720
2. **Browser**: Chrome in fullscreen (F11)
3. **Wallet**: Pre-connected, don't show setup
4. **Game State**: Start fresh, or at specific turn

### Recording Tools
- **macOS**: QuickTime Screen Recording
- **Windows**: OBS Studio or Xbox Game Bar
- **Online**: Loom (easy, auto-uploads)

### Before Recording
1. Clear browser console (no errors visible)
2. Close unnecessary tabs
3. Turn off notifications
4. Test audio levels
5. Practice run-through (timing)

### During Recording
1. Speak clearly and enthusiastically
2. Move cursor deliberately (not too fast)
3. Pause between sections (easier to edit)
4. If you mess up, just pause and restart that section

### After Recording
1. Trim dead space at start/end
2. Add background music (optional, low volume)
3. Add text overlays for key points (optional)
4. Export as MP4 (H.264)
5. Upload to YouTube or Loom
